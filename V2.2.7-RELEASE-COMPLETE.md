## Notable fixes shipped in v2.2.7

See `RELEASE-NOTES-v2.2.7.md` for the full list. Highlights:

* Windows: correct `HOME` / `PATH` / `PATHEXT` handling, `.exe` suffix on
  respawn, cmd.exe-aware install detection.
* Release pipeline: binary version audit gate, per-target builds, out-of-band
  SHA256 manifest publishing, testGate integrated with workspace tests.
* TUI: scrollback + text-selection pass-through via Shift-drag mouse events.
* QMD: cross-platform discovery; no longer hard-coded to Unix socket paths.
* Ollama: multi-format tool-use parser (Anthropic / OpenAI / XML / JSON-fence /
  natural-language) with fail-loud on ambiguous responses.
* Remote-control relay: `anvil-relay-ws` now in `ecosystem.config.cjs`; 503s
  resolved.
* Cross-OS installers: `install.sh`, `install.ps1` with SHA256 verification and
  dual-source fetch.
* `anvil --check` / `--setup` / `--uninstall` / `upgrade` subcommands.
* Shell completions (bash / zsh / fish / PowerShell) for 101 slash commands.
* Setup wizard: curated Ollama model menu (Llama 3.x, Qwen 3 / 2.5-Coder,
  Mistral Nemo / Codestral, Gemma 3, Phi 4, Code Llama) with per-model
  confirmation. No DeepSeek, no Kimi.
* Credential hygiene: API key prompt via `rpassword` (no echo), `chmod 0600` on
  `config.json` at write time, honest documentation of plaintext-at-rest with a
  pointer to vault migration.

## Release-gate fixes made during the final approach

Two parallel-test races that the hardened release MCP's test gate caught — both
fixed in commit `d9ae8c3`:

1. `permission_memory::tests` were writing to the real `~/.anvil/` because
   nothing redirected `ANVIL_CONFIG_HOME`. Replaced with a `ConfigHomeGuard`
   that points the env var at a per-test `TempDir` under the crate-wide
   `test_env_lock()` so `oauth` / `prompt` / `permission_memory` tests
   serialise their env mutation.
2. `mcp_stdio::tests::temp_dir()` generated paths with only a nanosecond
   suffix. Under load, two spawns in the same nanosecond collided and one
   python subprocess would lose its script file mid-startup. Path now
   incorporates PID + an atomic counter.
