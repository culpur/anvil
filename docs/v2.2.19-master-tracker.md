### Arc H — Live `t()` re-render in viewer

Viewer locale switch currently reloads the page (Phase W4 trade-off). v2.2.19 polish: make every `t()` callsite reactive so locale change re-renders in place without losing scroll position, in-flight tabs, or terminal state.

**Approach**: wrap `t()` calls in a tiny reactive layer — DOM nodes that contain translated text get a `data-i18n-key` attribute; on locale change, walk the DOM and re-translate all matched nodes. Status footer, tab labels, modal titles, terminal headers all live-update. SSH terminals keep their connection state.

**Acceptance**: open viewer, start an SSH tab, switch from English to Spanish via dropdown — terminal stays connected, labels around it change to Spanish, no page reload, no scroll-position loss.
